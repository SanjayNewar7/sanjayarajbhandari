
  # Graphics Designer Portfolio Website

  This repository contains the original design and development work of Sanjay Rajbhandari.
  The portfolio is crafted to showcase creative design skills, visual storytelling, and modern UI aesthetics suitable for a professional graphics designer.

  The design was conceptualized and designed by Sanjay Rajbhandari and is available on Figma:
  https://www.figma.com/design/fh5AKw2HCDRqqvm0jlRnfJ/Graphics-Designer-Portfolio-Website

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

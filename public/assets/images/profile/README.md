# Profile Images and Icons

This directory contains profile images and software icons for the portfolio website.

## Required Icons

The following icon files should be placed in this directory:

- `photoshop-icon.png` - Adobe Photoshop official icon
- `illustrator-icon.png` - Adobe Illustrator official icon
- `figma-icon.png` - Figma official icon
- `indesign-icon.png` - Adobe InDesign official icon
- `premiere-icon.png` - Adobe Premiere Pro official icon
- `coreldraw-icon.png` - CorelDRAW official icon

## Suggested Dimensions

- Profile image: 1080x1080 pixels (for the hero section)
- Software icons: 128x128 pixels (will be scaled down as needed)

## Note

For the software icons, please download the official SVG or PNG icons from the respective software companies' websites or use reputable icon libraries that provide official software icons. Make sure the icons are properly licensed for your use.
